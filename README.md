
# @peoro/eslint-config

> peoro's ESLint config

## Installation

```bash
npm install --save-dev @peoro/eslint-config
```

## Usage

Set your `.eslintrc.*`'s `extends` field to `"@peoro/eslint-config"`:

```json
{
	"extends": "@peoro/eslint-config"
}
```

That's it, ESLint will now use this config.
